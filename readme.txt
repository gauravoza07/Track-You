Tracks user activities including image uploads and ACF field changes with IP address logging.

== Description ==
Track You - User Activity Logger records key user actions like image uploads and ACF field updates, storing IP addresses for audit trails. Useful for debugging and compliance.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/track-you` directory or install via the WordPress Plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
